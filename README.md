# receiptSplitter
upload receipt, assign people, get the split payments at the end for splitwise 
1. adds people to split grocery bill based off of the items that they claim on the bill, one item can be claimed by one or more people 
2. scan receipt from pdf/screenshot 
3. assign to each item for every person, cascade style so it goes item by item in the list shown in the receipts. some receipts might have discounts for the item in the next line, please analyze correctly
4. split tax
5. email excel file with all relevant details
